Experiment 9: cflib Programming

cflib 库编程

Source-based English document

This page is an objective English document derived from the corresponding Word manual in the local source folder. The Chinese page keeps the extracted original text, tables, links, commands, and figures.

Purpose

Use cflib to structure Crazyflie connection, logging, parameter access, and command scripts.

Preparation

  • Working Crazyflie connection.
  • Python environment with cflib installed.
  • Known radio URI or scanning procedure.

Procedure

  1. Initialize CRTP drivers and create a Crazyflie object.
  2. Use synchronized connection patterns for safer program structure.
  3. Read parameters or logs and send controlled commands.

Verification

  • The script connects and exits cleanly.
  • Logged or printed values match the expected aircraft state.
  • Motion commands are short, conservative, and explainable.
