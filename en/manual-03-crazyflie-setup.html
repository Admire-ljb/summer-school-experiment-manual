Experiment 3: Initial Crazyflie Configuration and Operation

初次配置与驱动 Crazyflie 无人机

Source-based English document

This page is an objective English document derived from the corresponding Word manual in the local source folder. The Chinese page keeps the extracted original text, tables, links, commands, and figures.

Purpose

Prepare the Crazyflie software and hardware connection path for safe first operation.

Preparation

  • Ubuntu/ROS environment from the previous manuals.
  • Crazyflie aircraft, battery, USB cable, and Crazyradio.
  • cflib, Crazyflie client, and required Python dependencies.

Procedure

  1. Install the Crazyflie client and required libraries.
  2. Configure USB permissions and radio connection settings.
  3. Connect to the aircraft and run conservative first-control tests.

Verification

  • Crazyflie client starts correctly.
  • USB and radio connection can detect the aircraft.
  • A short scripted test can be explained and stopped safely.
