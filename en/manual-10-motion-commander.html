Experiment 10: Advanced Motion Commander Programming

Motion Commander 进阶编程

Purpose

Use Motion Commander movement primitives to construct repeatable flight routines.

Preparation

  • cflib script template.
  • Safe test area.
  • Instructor-approved height and speed parameters.

Procedure

  1. Create a MotionCommander context.
  2. Combine takeoff, landing, directional movement, turns, and pauses.
  3. Test short movement segments before a complete sequence.

Verification

  • The drone completes the intended primitive sequence.
  • Timing, height, and distance choices are recorded.
  • The program can be stopped safely if behavior deviates.
