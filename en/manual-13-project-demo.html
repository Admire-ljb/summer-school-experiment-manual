Experiment 13: Integrated Project Demonstration

综合项目展示任务

Purpose

Present a complete project task using the course experiments as technical basis.

Preparation

  • Project objective and success criteria.
  • Working code and tested hardware setup.
  • Presentation evidence such as logs, route sketches, video, or screenshots.

Procedure

  1. Select a feasible demonstration task.
  2. Implement and rehearse the workflow under safety constraints.
  3. Present the method, result, limitations, and improvement plan.

Verification

  • The demonstration is executable or reproducible.
  • The presentation explains method and evidence objectively.
  • Safety, teamwork, and failure handling are included in the report.
