Experiment 8: Path-planning Simulation

路径规划仿真

Purpose

Run path-planning simulation experiments and compare route-generation behavior.

Preparation

  • Simulation project files.
  • Ubuntu environment with required dependencies.
  • Terminal access in the project directory.

Procedure

  1. Prepare and build the simulation project as documented.
  2. Set start and goal conditions in the simulator or visualization tool.
  3. Run A*, RRT, RRT*, or the provided planning examples and observe outputs.

Verification

  • A route is generated for the selected task.
  • The observed route can be compared across algorithms or parameters.
  • Execution notes include commands, errors, and screenshots where applicable.
